# Tavern App — AI Runtime Platform 项目报告

## 概述

Tavern App 是一个 **AI Runtime Platform**，基于 Next.js 15 + React 19 + TypeScript 构建。它从 AI 角色扮演聊天应用起步，已演进为支持 **可组合 Workflow、图执行引擎、Agent 决策、目标驱动行为、持久化会话和运行时检查器** 的完整 AI 运行时平台。

**技术栈**: Next.js 15 · React 19 · Zustand · zod · Tailwind CSS 4 · DeepSeek API

## 数据流

```
用户输入
  │
  ▼
chatStore.sendMessage()
  │
  ├── SessionStore: appendUserMessage()
  │     └── SessionMemory: 自动提取记忆
  │
  ├── CapabilityComposer: Profile → Graph
  │
  ├── GraphExecutor.execute()
  │     ├── 拓扑排序
  │     ├── 每个节点: Zod校验 → 语义条件 → 执行
  │     ├── RuntimeSnapshot: 记录输入/输出/决策
  │     └── 返回: output + traces + snapshots
  │
  ├── SessionStore: appendAssistantMessage()
  │
  └── UI: RuntimeConsole / GraphInspector / TokenInspector
```
